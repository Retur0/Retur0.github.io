MySQL下数据备份和恢复

发表于 2019-10-21 分类于 数据库 Valine：

生成SQL脚本 导出数据

不需要登陆数据库，在控制台使用mysqldump命令可以生成指定数据库的脚本文本，但脚本文本中只包含数据库中的内容，而不会存在创建数据库的语句。所以在恢复数据时，还需要手动创建一个数据库之后再去恢复数据。放在C盘根目录下需使用管理员权限。

mysqldump -u用户名 -p密码 数据库名称>存放目录

执行SQL脚本 恢复数据

  1. 登陆数据库。

  2. 创建数据库。

  3. 执行sql脚本。

    SOURCE 脚本目录;

    mysql -u用户名 -p密码 数据库名称<要执行的脚本的路径;

也可以不登陆mysql：

mysql -u用户名 -p密码 数据库名称<要执行的脚本的路径

这个方法也需要先创建数据库。
